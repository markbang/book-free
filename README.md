# book-free
🚀推荐一些查找电子版图书的网站(Recommend some websites to find electronic books)

欢迎Issue推荐

### 神中神🔥🔥🔥

- [Z-Library](https://z-library.se/)  99%都能搜到，而且种类齐全
- [书栈网](https://www.bookstack.cn/)  编程书大全，分类很广，在线阅读
- [UP云搜](https://www.upyunso.com/)  阿里云网盘搜索引擎，内容很广，拿来搜书也不错
